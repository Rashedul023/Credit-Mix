# Credit Mix Prediction

## Overview
This project predicts customer credit mix (Poor, Standard, Good) using financial data and provides recommendations to improve credit health.

## Dataset
- 50,000 customer records
- Features: Age, Income, Loans, Payment History, Credit Utilization
- Target: Credit_Mix (3 categories)

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Results
Random Forest performed best with highest accuracy and F1-score.

## Key Findings
Top factors affecting credit mix:
1. Interest Rate
2. Outstanding Debt
3. Number of Delayed Payments
4. Payment of Minimum Amount
5. Number of Bank Accounts
